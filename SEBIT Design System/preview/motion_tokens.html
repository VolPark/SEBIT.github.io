> Motion · tokens & patterns

Pohyb se chová jako značka.
Snap, ne float.

SEBIT motion je rychlý, rozhodný a přesný. Žádné „zhouravé" bounce efekty, žádná 800ms transition. Cílem je potvrzení akce — ne show.

Duration tokens

5 hodnot pokryje 98 % UI. Když sáhneš po jiné, zpochybni to. Rule of thumb: malý prvek → krátké trvání; velká plocha → delší.

--dur-instant 80ms
Mikro feedback. Hover, focus ring, button press.
--dur-fast 160ms
Výchozí UI. Buttony, toggly, dropdowny, toast in.
--dur-base 240ms
Standard. Panely, karty, tab content, menu.
--dur-slow 400ms
Expresivní. Modaly, drawery, page transitions.
--dur-slowest 640ms
Choreografie. Onboarding, hero, data reveal.

Easing curves

Každý easing má jeden úkol. Nekombinuj ease-standard s ease-exit v jednom komponentu — matou.

--ease-standard 0.2, 0, 0, 1
Výchozí enter. Prvek přijíždí a rychle se zabrzdí. Buttony, karty, panely.
--ease-emphasized 0.3, 0, 0, 1
Hero enter. Silnější decelerace. Modaly, velké karty, onboarding kroky.
--ease-exit 0.4, 0, 1, 1
Odchod ze scény. Akcelerace — prvek zrychluje a zmizí. Dismiss toastů, zavírání menu.
--ease-in-out 0.4, 0, 0.2, 1
Symetrický pohyb. Přesuny A → B kde začátek i konec mají stejnou váhu. Accordiony, tab indicator.
--ease-signature SEBIT
Signature arrow snap. Ostré zastavení — stejná křivka jako v loaderu. Hero, splash, arrow marchy, loading.
linear —
Jen pro progres. Loading bar, skeleton shimmer, sparkline běh. Nikdy ne na UI prvcích.

Pravidla — co nedělat

✓ Dělej
  • Animuj jen transform a opacity — jinak trpí frame rate.
  • Drž se 5 durations a 5 easings. Vše ostatní je výjimka co musíš obhájit.
  • Entry používá standard/emphasized, exit používá exit. Odchod je rychlejší než příchod.
  • Respektuj prefers-reduced-motion — pak všechny animace → instant fade.
✕ Nedělej
  • Žádné bounce / elastic easings. SEBIT není hračka.
  • Žádné transitions > 640ms na interakční prvky. Uživatel čeká.
  • Žádný parallax, floating objects, perpetuálně pohybující se pozadí. Odpuzuje.
  • Žádné animované SVG ikony „pro efekt". Šipka je jediná výjimka.
  • Nikdy transition: all. Explicitně vypisuj property.
◐

Accessibility · reduced motion

Každá animace v produktu musí respektovat @media (prefers-reduced-motion: reduce). V tomto režimu: všechny durations → --dur-instant, všechny transforms → jen opacity fade. Marching arrow, draw loader a snap přechody jsou vypnuté — nahradí je statický glyph.

Implementace · CSS tokeny ke zkopírování

/* === MOTION TOKENS · sebit.css === */
:root {
/* Duration */
--dur-instant: 80ms;
--dur-fast: 160ms;
--dur-base: 240ms;
--dur-slow: 400ms;
--dur-slowest: 640ms;
/* Easing */
--ease-standard: cubic-bezier(0.2, 0, 0, 1);
--ease-emphasized: cubic-bezier(0.3, 0, 0, 1);
--ease-exit: cubic-bezier(0.4, 0, 1, 1);
--ease-in-out: cubic-bezier(0.4, 0, 0.2, 1);
--ease-signature: cubic-bezier(0.7, 0, 0.3, 1);
}
/* Příklad — button hover */
.btn { transition: background var(--dur-fast) var(--ease-standard),
transform var(--dur-fast) var(--ease-standard); }
