Emailový podpis · univerzální responsive

Jeden HTML blok, který se sám přeskládá na mobilu.

Jak to funguje: desktop zobrazení používá tabulku se dvěma buňkami vedle sebe. Na obrazovkách do 480 px se přes @media buňky přeskládají pod sebe. Outlook pro Windows media queries ignoruje — zůstane desktop varianta (což je v pořádku, Outlook Win je desktop klient).

HTML kód — jeden blok pro oba pohledy

Nasazení v klientech

  • Outlook (Windows / Mac desktop): Soubor → Možnosti → Pošta → Podpisy → Nové. Vložte HTML přes pravý klik → Vložit zdrojový kód. Outlook zobrazí vždy desktop variantu.
  • Outlook Web / Microsoft 365: Nastavení → Email → Podpisy. Vložte rendered podpis přímo (tlačítko „Kopírovat rendered"). Media queries fungují — podpis se příjemci na mobilu přeskládá.
  • Gmail (web): Otevřete tento soubor v prohlížeči, označte náhled desktopu, Cmd/Ctrl+C, vložte do Nastavení → Podpis. Nebo použijte tlačítko „Kopírovat rendered".
  • Apple Mail (macOS / iOS): Podporuje media queries. Podpis bude responsive automaticky.
  • Gmail app (iOS / Android): Nastavte v Gmail webu — app to zdědí. Media queries fungují.
