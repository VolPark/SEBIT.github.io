# SEBIT Marketing UI Kit

A pixel-faithful recreation of the SEBIT Solutions marketing site (`sebit.cz`) as
modular JSX components. Uses Tailwind CDN + our `colors_and_type.css` tokens.

## Components
- `Nav.jsx` — fixed white header with logo + menu
- `Hero.jsx` — navy hero with headline, subtitle, pill CTA and arrow signet
- `DivisionRow.jsx` — four-tile cross-link to Solutions/Labs/Medical/Academy
- `ServiceCard.jsx` — white card with hover-lift + "Zjistit více"
- `StatBlock.jsx` — lime numeral + gray caption for reference section
- `Testimonial.jsx` — translucent card on navy
- `ContactForm.jsx` — two-column contact section (details + form)
- `Footer.jsx` — navy footer with LinkedIn + copyright

`index.html` wires them into a working clickthrough of the Solutions homepage.
